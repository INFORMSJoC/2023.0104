Extending the libraryÔÉÅ

It is possible to write custom implementations of the routingblocks.PyEvaluation and routingblocks.Propagator classes. The native extensions github template provides the necessary boilerplate code along with an example to get started.

The source code of routingblocks.adptw.Evaluation (native/src/ADPTWEvaluation.cpp), routingblocks.niftw.Evaluation (native/src/NIFTWEvaluation.cpp), routingblocks.adptw.FRVCP (native/include/routingblocks/ADPTWEvaluation.h), and routingblocks.niftw.FRVCP (native/include/routingblocks/NIFTWEvaluation.h) provides further examples.
